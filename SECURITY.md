# Security Policy

## Supported Versions

This repository maintains the currently deployed version of the website.
Only the latest production version is supported for security updates.

## Reporting a Vulnerability

If you find a security issue related to this website, please report it privately by email instead of creating a public issue.

Contact: [borim0810@gmail.com]

When reporting, please include:
- A clear description of the vulnerability
- The URL or page where it occurs
- Steps to reproduce the issue
- Any supporting material such as screenshots or proof-of-concept code

I will review reports as soon as possible and aim to respond within 3–7 days.

Please avoid disclosing the issue publicly until it has been reviewed and addressed.

Thank you for reporting security issues responsibly.
